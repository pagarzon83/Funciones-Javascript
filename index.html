Funciones en JavaScript

Funciones Declaradas

Las funciones declaradas son aquellas que se definen utilizando la palabra clave "function".

Su estructura básica es la siguiente:


                function nombreFuncion(parametros) {
                    // Código a ejecutar
                    return resultado;
                }

Algunas ventajas de las funciones declaradas son:

  • Son hoisted, lo que significa que se pueden llamar antes de ser definidas en el código.
  • Se pueden llamar desde cualquier lugar del código.

Un ejemplo de función declarada sería:


                function suma(a, b) {
                    return a + b;
                }
                console.log(suma(2, 3)); // Output: 5

Funciones Expresadas

Las funciones expresadas son aquellas que se definen como una expresión, es decir, asignando la función a una variable.

Su estructura básica es la siguiente:


                let nombreFuncion = function(parametros) {
                    // Código a ejecutar
                    return resultado;
                }

Algunas ventajas de las funciones expresadas son:

  • No son hoisted, lo que significa que deben ser definidas antes de ser llamadas.
  • Permiten crear funciones anónimas.

Un ejemplo de función expresada sería:


                let resta = function(a, b) {
                    return a - b;
                };
                console.log(resta(5, 2)); // Output: 3

Funciones de Flecha

Las funciones de flecha son una forma simplificada de escribir funciones expresadas.

Su estructura básica es la siguiente:


                let nombreFuncion = (parametros) => {
                    // Código a ejecutar
                    return resultado;
                }

Algunas ventajas de las funciones de flecha son:

  • Son más cortas y fáciles de leer.
  • Útil para definir funciones cortas y simples.

Un ejemplo de función flecha sería:


                let multiplicacion = (a, b) => {
                    return a * b;
                };
                console.log(multiplicacion(2, 4)); // Output: 8
